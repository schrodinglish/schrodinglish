### ⌨️ Git Commands
| Command | Explanation |
|---------|-------------|
| `git clone <url>` | Copy a remote repo to your local machine |
| `git add .` | Stage all changes, ready to be committed |
| `git commit -m "message"` | Save a snapshot of your changes with a note |
| `git push` | Upload your local commits to GitHub |
| `git pull` | Download the latest changes from GitHub |

> Clone once, then add → commit → push every day.
