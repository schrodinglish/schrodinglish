## 📝 Markdown Syntax

### Headings
````md
# H1
## H2
### H3
````

### Text Styling
````md
**bold**
*italic*
~~strikethrough~~
`inline code`
````

### Lists
````md
- unordered item
- unordered item

1. ordered item
2. ordered item
````

### Links & Images
````md
[link text](https://url.com)
![alt text](image-url.png)
````

### Blockquote
````md
> This is a quote
````

### Code Block
````md
```javascript
console.log("hello world");
```
````

### Table
````md
| Header | Header |
|--------|--------|
| cell   | cell   |
````

### Horizontal Rule
````md
---
````
